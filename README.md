This portfolio landing page brings together the web projects I have been developing as part of my ongoing front-end learning.
My initial project, the flag finder tool, uses JavaScript to communicate with an external API. When a user types a countries name
into the search box, the site sends a request to a third-party data source, retrieves the relevant information, and displays the flag 
of the searched country on the page. It has a simple design, but was a great test for me when working with DOM manipulation and an API.

The second project, Studio-Celest, was built collaboratively with a course colleague. Together, we created a mock clothing store 
that features a functional shopping basket, powered by JavaScript, and supported by image folders for product data. I handled the 
homepage layout, visual design, and overall UI interface, working on structure and styling across the multiple page website. This 
project was a genuinely rewarding experience, and was an opportunity collaborate with a team in GitHub. Creating a fun brand by 
combining design and functionality to bring the concept to life.
